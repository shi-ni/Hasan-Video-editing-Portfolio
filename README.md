## About This Project

This is a custom one-page portfolio website for Hasan Zubair, designed to present video editing work with a cinematic visual style.

The site includes:

- Hero section with branded introduction and call-to-action buttons
- About section with experience stats and editing specialties
- Embedded video showcase for landscape and short-form projects
- Fiverr service cards for direct client conversion
- Responsive layout optimized for desktop and mobile

## Creative Direction

- Mood: cinematic, elegant, premium
- Visual language: black and gold palette with subtle film-grain texture
- Typography: Cormorant Garamond + Outfit
- Interaction: soft entrance animations and polished hover states

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages for hosting
- GitHub Actions for automatic deployment

## Local Development

1. Clone the repository.
2. Open the project folder.
3. Launch index.html in a browser.

## Deployment

Deployment is fully automated with GitHub Actions.

1. Push changes to main branch.
2. GitHub Actions runs deploy-pages.yml.
3. Site updates automatically on GitHub Pages.
